## DB Crud Operations

### Features -
1. **Add Product**
2. **Display Products**
3. **Edit Product**
4. **Delete Product**
5. **Search**

### Database Setup
- Database Name: `product_db`
- Table Name: `products`
- Import the `products.sql` file to set up the database structure.

### Configuration
- Database connection settings are in `db.php`.
- Default credentials used: User `root`, Password `""`.

### Files
- `add_product.php`: Create new.
- `display_products.php`: Read and display.
- `edit_product.php`: Update existing.
- `delete_product.php`: Delete.
- `search_products.php`: Search.
- `search_ajax.php`: Backend for search.
